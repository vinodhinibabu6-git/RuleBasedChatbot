# Rule-Based Chatbot

A simple chatbot developed using Python that responds to user inputs based on predefined rules.

## Features
- Greets the user
- Answers simple questions
- Uses if-else statements and pattern matching
- Exits when the user types "bye"

## Technologies Used
- Python
- Conditional Statements
- Loops
- String Handling

## How to Run

1. Clone the repository.
2. Open terminal.
3. Run:

python chatbot.py

## Sample Output

You: hello
Bot: Hello! Nice to meet you.

You: python
Bot: Python is a programming language.

You: bye
Bot: Goodbye!
